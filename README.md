# OpenInnocent

## Description
This project focuses on the **decompiling and native PC engine reimplementation** of the game "Innocent Life: A Futuristic Harvest Moon" (originally for PSP). 

The goal is to provide a modern, open-source framework that allows the game to run natively on PC, enabling higher resolutions, modding support, and improved performance.

## Status
- [ ] Reverse Engineering & Asset Extraction
- [ ] Native Engine Development
- [ ] Script Interpretation & Logic Porting

## Legal Notice
This project is an open-source engine reimplementation. It **does not contain** any copyrighted assets or game files. To run this project, users must provide their own legitimate backup of the original game files.
